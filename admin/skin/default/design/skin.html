  • Q. FTP 접속 후 상품디스플레이 파일을 수정하였습니다. 그리고 쇼핑몰 화면을 확인하였는데 수정이 되지 않았습니다.

  • A. 쇼핑몰화면에서 상품디스플레이 영역을 빠르게 로딩하기 위하여 캐시 기능을 적용되어 있기 때문입니다.

    아래와 같이 FTP로 접속하여 상품디스플레이 파일을 직접 수정 하실 경우에는

    상품디스플레이 영역의 캐시 기능을 OFF한 후에 수정된 화면을 확인해 주십시오.

    1) FTP 접속

    2) data/skin/스킨명/_modules/display/ 디렉토리 안에 있는 파일 수정

    (data/skin/스킨명/_modules/display/ 디렉토리 안에 있는 파일이 상품디스플레이 파일입니다)

    3) 쇼핑몰 관리자환경 > 디자인 > 스킨 설정 → [상품디스플레이 영역 캐시 OFF하기] 클릭

    4) 쇼핑몰 화면에서 수정된 상품디스플레이 확인

    5) data/skin/스킨명/_modules/display/ 디렉토리 안에 있는 파일 수정이 완료되면

    쇼핑몰 관리자환경 > 디자인 > 스킨 설정 → [상품디스플레이 영역 캐시 ON하기] 클릭하여 상품디스플레이 로딩 속도를 다시 빠르게 함

  • ▶ 상품디스플레이 영역 캐시 기능 : 현재 { ? cfg_system.display_cach == 'OFF'}OFF{ : }ON{ / } 상태
