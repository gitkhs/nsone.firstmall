쿠폰 사용 기간 :   -     

1. 주문내역서에 출고번호를 바코드로 출력하시겠습니까?
출고 검색창에서 바코드를 스캔하면 해당 출고건의 출고화면으로 바로 이동하여 출고처리가 편리해집니다.
 

2. 출고내역서에 출고상품의 상품코드를 출력하시겠습니까?
 

3. 출고내역서에 출고상품의 상품코드를 바코드로 출력하시겠습니까?
해당 출고의 출고처리화면에서 실제 상품의 바코드를 스캔하면 출고상품이 맞는지 검증하여
오배송 없이 정확하게 출고가 가능합니다. 해당 상품의 바코드를 계속 스캔하면 출고수량이 +1씩 증가합니다.
 

외부 판매마켓에서 발생한 주문은
1시간에 2번씩 자동으로 수집합니다.
자동으로 수집되는 시간을 기다리기 힘드시면
[지금바로 주문수집] 버튼을 클릭하십시오.
