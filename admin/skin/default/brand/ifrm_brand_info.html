브랜드 기능 안내
브랜드 분류를 통하여 해당 상품을 체계적으로 관리하게 됩니다.
상품상세페이지에서는 대표브랜드가 해당 상품의 브랜드로 보여지며,
브랜드 페이지에서는 해당 브랜드에 속한 상품들이 보여집니다.

브랜드 기능은 '브랜드'란 단어에 국한될 필요 없이 자유롭게 이용할 수 있습니다.
예를 들어 도서 같은 경우 저자나 출판사의 분류 체계나
신상품, 베스트상품 등의 분류 체계로 활용하셔도 됩니다.
물론 운동화 브랜드(나이O, 아디다O, 뉴발란O)의 분류 체계로 활용하셔도 됩니다.

(브랜드 기능 활용 예시)
- 저자 : 다양한 저자명
- 출판사 : 다양한 출판사명
- 운동화브랜드 : 다양한 브랜드명
- 가격대별상품 : 다양한 가격대
- 기타 : 신상품, 베스트상품, 인기브랜드, MD추천, 이벤트상품

조건1) 선택된 회원등급만 접속을 허용

조건2) 선택된 회원 유형만 접속을 허용

조건1과 조건2를 모두 만족해야만 접속이 가능합니다.
